<h1>Windows Server 2022 Setup</h1>

<br />

<h3>Download a Windows Server 2022 64‑bit ISO from the Microsoft website.</h3>
<br />

<p align="center">
  <img src="https://imgur.com/O1Klayb.png" width="85%" />
</p>

<br />
<br />

<h3>In VirtualBox, press New at the top left.</h3>
<br />

<br />
<br />

<h3>Name it Windows Server 2022.</h3>
<br />

<p align="center">
  <img src="https://imgur.com/Z3V6cCm.png" width="85%" />
</p>

<br />
<br />

<h3>Click the ISO image box and press Other, then select the ISO you downloaded from the Microsoft website.</h3>
<br />

<p align="center">
  <img src="https://imgur.com/8jVBfIx.png" width="85%" />
</p>

<br />
<br />

<h3>Allocate 8 GB RAM, 2 CPUs, and 100 GB of storage.</h3>
<br />

<br />
<br />

<h3>Create it and boot it.</h3>
<br />

<br />
<br />

<h3>Choose Desktop Experience for a GUI.</h3>
<br />

<p align="center">
  <img src="https://imgur.com/1W0cHJi.png" width="85%" />
</p>

<br />
<br />

<h3>Choose Custom, then Next.</h3>
<br />

<p align="center">
  <img src="https://imgur.com/NYdU6kS.png" width="85%" />
</p>

<br />
<br />

<h3>Press Restart Now.</h3>
<br />

<br />
<br />

<h3>Choose a password.</h3>
<br />

<br />
<br />
